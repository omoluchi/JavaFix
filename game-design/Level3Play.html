Starting with Player 1, everyone takes a turn to describe a memory they have that surrounds their Object Gamma—without describing the object itself.   1. An emotion that comes to mind when hearing the memory recounted—how did the story make you feel?
  2. What fantastic contraption or object that might reproduce that same emotion in their Agent's universe.
